# Dopamine-Guard
Système de régulation de transe cognitive sous Linux (Ubuntu 24.04).

## Compilation
Pour compiler le moteur C, exécutez la commande suivante dans le terminal :
`make`

## Utilisation
1. Lancer le moteur C : `./moteur_muscle`
2. Lancer le capteur IA : `python3 ia_capteur.py`

## Dépendances
- OpenCV, MediaPipe (Python)
- libx11-dev, libxxf86vm-dev (C)
